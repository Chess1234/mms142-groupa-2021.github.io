What is Edge Computing

Edge Computing is an "alternative" way to compute, store, and process data. It is set in between the device and the cloud to help lessen latency and increase computing and analysis speed.

“Edge computing brings processing capabilities closer to the end-user/device/source of data which eliminates the journey to the cloud data centre and reduces latency.” - Dalia Adib

BENEFITS OF EDGE COMPUTING

    Speed - Multiple localized micro data centers = less reliance on cloud data centers which avoids bandwidth traffic, latency issues, etc.
    Security - Decentralized computing architecture makes it harder for attacks to be devastating (malware, phishing, DDOS) as the processes, storages, and applications are spread across multiple localized data centers.
    Scalability - It is not demanding in terms of resources compared to having to maintain a fully-pledged data center that needs proper maintenance and substantial bandwidth allocation.
    Versatility - By having edge data centers, it is possible to make changes depending on the end-users preference or target a market trend without completely altering the IT infrastructure.
    Reliability - Edge computing chases almost all loose ends of a centralized data center. It reduces the risk of complete collapse of the server in case of an outage, data can be rerouted through multiple gateways that increases the speed of which you get the information and it increases efficiency of data exchange between two machines.

EDGE VS CLOUD

Cloud computing in the most simplest form is any computing processes that are done via the internet. All the data are accessed, stored, and processed through the web which are all located in a massive storage system of data called data centers.
Edge computing on the other hand, can be seen as sort of the "middleman" that places all the computing resources closer to the user instead of having to travel from the main server and back which can cause latency issues. It is like storing it in a cache memory for later use. Edge computing is a game-breaking concept that when it is implemented alongside what we have now which is cloud computing, can solve a lot of security risks, delays, and better user experience.
